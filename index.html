Bem-vindo ao Meu Site

O que é HTML?

HTML (HyperText Markup Language) é a linguagem base da web.
Ela estrutura páginas com títulos, parágrafos, links e imagens.
Com HTML, você dá forma ao conteúdo exibido no navegador.
É o primeiro passo para criar sites profissionais.

Para que servem as tags HTML?

As tags HTML definem a função de cada elemento na página.
Com elas, você insere textos, listas, links e muito mais.
Cada tag tem um papel específico, como <h1> para títulos ou <a> para links.
Aprender as tags é essencial para quem está começando no front-end.

Sobre os streamings

Hoje em dia, os serviços de streaming dominam o entretenimento.
Netflix, Amazon Prime, Disney+ e HBO Max lideram o mercado.
Cada um oferece conteúdos exclusivos e planos acessíveis.
A escolha ideal depende do seu gosto e do catálogo favorito.

Comparando os principais streamings

Para quem gosta de filmes e séries variados, a Netflix é destaque.
Disney+ brilha com produções da Marvel, Star Wars e Pixar.
Já a Max atrai com séries consagradas e filmes recentes.
Compare os catálogos e escolha o que mais combina com você.
